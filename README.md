# devBlog

The Blog for Developers

## Stack

- Next.js
- Prisma
- GraphQl
- Chakra UI
- Postgresql
- Docker

## Run locally

Create a `.env.local` file based on `.env.local.sample`

To run this project using docker

```sh
docker compose up
```

or with older version

```sh
docker-compose up
```

Site will be available at http://devblog.localhost
